Tutorial

Cómo hacer una donación

¡Bienvenido al tutorial! Sigue estos pasos para completar tu donación.

Paso 1: Registro de usuario

Si no tienes una cuenta, regístrate antes de realizar una donación.

Paso 2: Inicia sesión

Ingresa con tu cuenta para acceder al formulario de donación.

Paso 3: Selecciona el tipo de donación

Elige entre textil, juguetes, productos de higiene o no perecederos.

Paso 4: Llena el formulario de donación

Proporciona los detalles de los artículos que donarás.

Paso 5: Selecciona un punto de recolección

Escoge una ubicación para entregar tu donación.

Paso 6: Guarda tu QR

Haz captura de pantalla a tu código QR.
